WHAT IS NIKE'S RETURN POLICY?

We give you 60 days to try out your QAZstore purchase to make sure it works exactly right for you.

If it’s not just right, here’s what you need to know:

  • You can return most items for any reason within 60 days of an online order delivery or a store purchase.
  • That includes custom all products.
  • Returns are free for QAZstore Members.
  • Courier delivery to DHL address.

So go ahead, shop with confidence and enjoy your 60-day trial.

TEMPORARY RETURN EXTENSION

Please note the 60-day return period represents a temporary extension of our return policy. Some exceptions apply.

RETURN POLICY EXCEPTIONS
  • Gift cards cannot be returned.
  • Items noted as nonreturnable cannot be returned.
  • Items purchased at QAZstore Clearance stores cannot be returned.
  • Shipping fees are not refundable unless you receive an incorrect item.
  • Some QAZstore products have specific written guarantees and/or warranties—please check the label, hang tag, or warranty card for details and return information.
  • QAZstore Clearance stores do not accept returns of any kind.
How do I return a QAZstore purchase?
You can take the items you want to return to any QAZstore, except QAZstore Clearance.
